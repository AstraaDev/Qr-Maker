# Qr Maker [![Build Status](https://img.shields.io/badge/covarage-100%25-succes)]()

> Convert a text/link into a Qr code and save it.

# Features
 - Graphic display.
 - Simple to use.

# How to use
 1. Install external modules.
 2. Launch [qr_maker.py](qr_maker.py).
 3. Enjoy !
